1. Материалы для отражательных спутников
Определена структура зеркала:
Солнечное излучение
↓
┌─────────────────────────────────┐
│  Al₂O₃ (защитный слой)                                                    │  
├─────────────────────────────────┤
│  Al (отражающий слой)                                                    │
├─────────────────────────────────┤
│  CVD SiC (полируемый слой)                                          │ 
├─────────────────────────────────┤
│  SiC основа (структурный слой)                                      │
└─────────────────────────────────┘
* В нашем проекте отражательные спутники оборудованы только пассивным охлаждением.
* Функции слоёв:
1. Структурный ー служит для термостабилизации зеркала, обеспечения жёсткости в условиях космоса.
2. Полируемый ー обеспечивает гладкость поверхности для нанесения отражательного слоя. (необходим, так как поверхность карбида кремния имеет шероховатости (2)
3. Отражательный ー непосредственно отражает излучение
4. Защитный ー необходим для защиты от атомарного кислорода, радиации и механических повреждений.
Выбор материала для защитного слоя обусловлен тем, что Al₂O₃ обладает превосходной радиационной стойкостью по сравнению с традиционным SiO₂.(3)
* В качестве альтернатив для отражающего слоя рассматривались: серебро (Ag), стекло, тугоплавкие металлы (молибден, тантал, вольфрам)
Критериями для выбора материала являлись:
1. Плотность (должна быть наименьшей для облегчения и удешевления доставки)
2. Отражательная способность (отношение коэффициента поглощения к коэффициенту отражения)
3. Устойчивость к деградации различного рода (термической, радиационной)
4. Масштабируемость


* Ключевым параметром деградации является рост коэффициента солнечного поглощения. В соответствие с исследованием выяснено, что основной вклад в изменение оптических свойств материала вносит воздействие протоновского излучения.

Расчёт массы материалов на м² зеркала:
1. Структурная основа – по данным космических зеркал (SOFIA, LORRI, Gaia) поверхностная плотность облегчённых SiC-зеркал составляет 16–17 кг/м². Принято 16.5 кг/м². 
Эффективная толщина сплошного слоя: 16.5 / 3210 ≈ 5.1мм.

2. Определение расстояния, на которое Рой будет удалён от солнца


Для минимизации экономических потерь очевидно, что необходимо располагать Рой Дайсона как можно ближе к солнцу.
Соответственно, единственным ограничивающим фактором в данном случае является температура, при которой отражатели смогут функционировать без существенной деградации составных элементов. 
В нашей структуре самым нагревающимся элементов является алюминий. За отсутствием экспериментальных данных по максимальной температуре, при которой материалы не испытывают существенной деградации, принята оценка в 350°C. Она учитывает: 
* Температуру плавления алюминия (660°C) — верхний физический предел
* Начало рекристаллизации и роста зёрен (150-200°C для тонких плёнок)
* Данные по диффузии Al в SiC (активируется выше 400°C)
* Инженерный запас прочности для длительных миссий
